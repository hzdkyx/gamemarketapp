# Documentação GameMarket API

A URL oficial `https://gamemarket.com.br/api-docs` retornou `403 Forbidden` quando acessada automaticamente neste ambiente.

Antes de implementar a Fase 4, salve aqui a documentação oficial da GameMarket, incluindo:

- autenticação;
- endpoints;
- webhooks;
- validação de assinatura/token;
- payloads reais;
- nomes reais de eventos;
- limites de rate limit;
- exemplos de erro.

Sem essa documentação, o projeto não deve inventar endpoints, payloads, autenticação ou nomes de eventos.
